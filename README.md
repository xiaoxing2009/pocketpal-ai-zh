# PocketPal AI 📱🚀

    - [iOS](#ios)

PocketPal AI 是一款口袋大小的 AI 助手，由运行在您手机上的小型语言模型（SLMs）驱动。PocketPal AI 专为 iOS 和 Android 设计，让您无需互联网连接即可与各种 SLMs 进行交互。您的隐私完全得到保护，因为所有处理都在设备上完全进行——您的对话、提示和数据永远不会离开您的手机或存储在外部服务器上。

## 功能
1. 确保已加载模型。
- **离线 AI 辅助**: 无需互联网连接，直接在您的设备上运行语言模型。
- **模型灵活性**：下载并切换多个 SLM，包括 Danube 2 和 3、Phi、Gemma 2 和 Qwen。
- **自动卸载/加载**：当应用处于后台时自动管理内存，卸载模型。
- **推理设置**：自定义模型参数，如系统提示、温度、BOS 标记和聊天模板。

- **实时性能指标**：在 AI 响应生成过程中查看每秒令牌数和每令牌毫秒数。

- **消息编辑**：编辑您的消息并重试 AI 生成。

- **个性化伙伴**：通过自定义设置创建不同 AI 个性。

- **后台下载**：在 iOS 设备上使用其他应用时继续下载模型。

- **推理时屏幕常亮**：在 AI 生成回复时保持屏幕开启。

- **多设备支持**: 专为手机和平板电脑优化，包括 iPad。

- **本地化**: 使用您偏好的语言。

- **Hugging Face 集成**: 通过身份验证访问公共和受限制模型。

1. 打开应用并点击**菜单**图标（☰）。
2. 进入**模型**页面。

3. 从列表中选择一个模型并点击**下载**。

1. 长按任意消息进行编辑。
2. 编辑后，AI 将根据您的修改重新生成其回复。
3. 使用重试选项，在不修改消息的情况下获取新的回复。

### 使用伙伴

1. 创建具有不同个性和设置的个性化 AI 助手。

- **角色扮演伙伴**: 类似于助手伙伴，并增加了位置、AI 角色和其他上下文参数的额外设置。
3. 使用聊天页面中的伙伴选择器选择一个伙伴，以便快速切换不同的角色。

## 开发环境设置

想参与贡献或本地运行应用？请按照以下步骤操作。

### 前置条件

- **Node.js**（版本 18 或更高）
- **Yarn**
- **React Native CLI**
- **Xcode** (用于 iOS 开发)
- **Android Studio** (用于 Android 开发)

### 开始使用

1. **Fork 并克隆仓库**

```bash
   git clone https://github.com/a-ghorbani/pocketpal-ai
   cd pocketpal-ai
   ```

2. **安装依赖**

   ```bash
   yarn install
   ```

3. **安装 Pod 依赖（仅限 iOS）**

   ```bash
   cd ios
   pod install
cd ..
   ```

4. **运行应用**

   - **iOS 模拟器**

```bash
yarn ios
```

- **Android Emulator**

```bash
yarn android
```

### 脚本

- **启动 Metro Bundler**

  ```bash
  yarn start
  ```

- **清理构建工件**

  ```bash
  yarn clean
  ```

- **代码风格检查和类型检查**

  ```bash
  yarn lint
  yarn typecheck
```

- **运行测试**

  ```bash
  yarn test
```
```

## 致谢

PocketPal AI 是基于以下出色工作构建的：

- **[llama.cpp](https://github.com/ggerganov/llama.cpp)**: 能够在本地设备上高效推理 LLMs。
- **[llama.rn](https://github.com/mybigday/llama.rn)**: 将 llama.cpp 绑定到 React Native 中。
- **[React Native](https://reactnative.dev/)**: 支持跨平台移动体验的框架。
- **[MobX](https://mobx.js.org/)**: 保持应用响应性和性能的状态管理库。
- **[React Native Paper](https://callstack.github.io/react-native-paper/)**: 用于 UI 的 Material Design 组件。
- **[React Navigation](https://reactnavigation.org/)**: 应用的屏幕路由和导航。
- **[Gorhom Bottom Sheet](https://github.com/gorhom/react-native-bottom-sheet)**: 驱动整个应用的流畅底部面板交互。
- **[@dr.pogodin/react-native-fs](https://github.com/birdofpreyru/react-native-fs)**: 文件系统访问，支持模型下载和管理。

以及许多其他开源库，使这个项目成为可能！
